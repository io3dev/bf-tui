# bf-tui
A brainfck interpreter with tui

![](https://github.com/io3dev/bf-tui/blob/main/images/Hello_World.gif)

## Building

Clone this repository and run the makefile, it uses g++ version 12.1.0

```shell
https://github.com/io3dev/bf-tui
cd bf-tui
make
```

And run it (on linux)

```shell
./bin/bf.out
```
